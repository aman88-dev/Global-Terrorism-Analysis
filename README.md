## Overview
This project performs Exploratory Data Analysis (EDA) on the United Nations Global Terrorism Analysis (UNGTA) dataset, which contains over 180,000 terrorist incidents recorded worldwide between 1970 and 2017. The objective is to identify patterns, trends, and key insights related to global terrorism.

---

## Objectives
- Analyze trends in terrorist incidents over time  
- Identify high-risk countries and regions  
- Examine attack types and weapon usage  
- Assess the impact of attacks in terms of casualties  
- Generate meaningful insights using visualizations  

---

## Tech Stack
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## Dataset Description
The dataset includes:
- Event details (date, location, attack type)  
- Geographic information (country, region, city)  
- Attack characteristics (weapon type, target type)  
- Impact metrics (number of killed and wounded)  

---

## Project Workflow

### 1. Data Loading and Inspection
- Imported dataset and reviewed structure, columns, and data types  

### 2. Data Cleaning
- Handled missing values  
- Removed inconsistencies  
- Selected relevant features  

### 3. Exploratory Data Analysis
- Time-based trend analysis  
- Country and region-wise analysis  
- Attack and weapon type distribution  
- Casualty impact analysis  

### 4. Visualization
- Line plots for yearly trends  
- Bar charts for country comparisons  
- Histograms for distribution analysis  
- Heatmaps for correlation  

---

## Key Insights
- Terrorist incidents are concentrated in specific regions  
- Significant increase in attacks observed after 2000  
- Bombings and armed assaults are the most common attack types  
- Explosives and firearms are the most frequently used weapons  
- A small number of incidents account for a large proportion of casualties  

---


## Limitations
- Missing or incomplete data in some fields  
- Dataset limited to the year 2017  
- Possible reporting bias across regions  

---

## Conclusion
This project demonstrates how exploratory data analysis can extract meaningful insights from large-scale real-world data. The findings help in understanding global terrorism patterns and support data-driven decision-making.
